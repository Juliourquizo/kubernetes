# Práctica kubernetes

## Prerequisitos

Descargar la imagen de Ubuntu para importarla en VirtualBox [Descargar](https://drive.google.com/file/d/17dI5sOsgoBoYH78P7VoDhdJl_6OSIGqY/view?usp=sharing)
